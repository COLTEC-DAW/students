Manual de programação

Introdução a linguagem C

Um programa em C é composto por um conjunto de funções. A função pela qual o programa começa a ser executado chama-se "main" que significa "principal". Um ponto-e-vírgula é colocado após cada comando do programa. Além disso, todo programa em C deve ser organizado para ser mais legível.

O que são variáveis?

Variável é uma posição de memória que pode ser identificada através de um nome. Elas mudam de valor após um comando de atribuição.

As variáveis podem ser utilizadas para realizar expressões matemáticas como soma, subtração, divisão, multiplicação, etc.

Exemplo de declaração de variável e expressão matemática

    #include<stdio.h>

    int main () {
        int a, b, soma; //declaração de uma variavel inteira
        a = 2;  //recebe o valor 2
        b = 4;  //recebe o valor 4
        soma = a + b;   //recebe a soma das variaveis anteriores
    }

Estruturas condicionais

Condição

Em C, uma condição nada mais é do que uma expressão cujo valor resultante precisa ser um número inteiro. Caso esse valor seja 0, a expressão é dada como falsa. Caso seja qualquer outro valor não-nulo, a expressão é dada como verdadeira. Uma estrutura condicional é representada pela variável "if" e, em alguns casos, "else".

Operadores de comparação

Os operadores de comparação são usados para comparar dois valores. São tratados da mesma forma que os operadores aritméticos.

Exemplo de estrutura condicional

    #include<stdio.h>
    
    int main () {
        int n;
        printf("Digite um valor para n\n");
        scanf("%d", &n);
        if (n>0) {
            printf("O número é positivo\n");
            else if (n<0) {
                printf("O número é negativo\n");
            else if (n=0) {
                printf ("O número é neutro\n");
                }
            }
        }
        return 0;
    }

Estruturas de repetição

Vetores

os vetores são uma estrutura de dados que podem armazenar uma determinada quantidade de valores do mesmo tipo.

O que é uma estrutura de repetição?

Uma estrutura de repetição é uma estrutura que permite executar mais de uma vez o mesmo comando ou conjunto de comandos, de acordo com uma condição ou contador. É geralmente iniciada pelas variáveis "while", "do - while" ou "for".

Exemplo de estrutura de repetição

    #include<stdio.h>
    
    int main() {

        int numero = 1;
        while (numero <= 10) { //faz o loop
            printf("%d\n" , numero); //imprime o numero
            numero = numero + 1; //atualiza o numero em 1 depois que o numero eh impresso
        }
    }

Como executar um programa em C?

Para executar um programa em C, caso o usuário tenha um programa externo que compila automaticamente, como o codeblocks, basta rodar o compilador. Caso o usuário não tenha esse tipo de programa, basta digitar no prompt de comando: o compilador (gcc), nome do seu código fonte.c -o (que é a saída) nome do código final, apertar enter e digitar: ./nome do código final.out (para executar o programa).

Exemplo de execução de código em C

gcc nomedomeucodigo.c -o nomedomeucodigofinal
./nomedomeucodigofinal.out
